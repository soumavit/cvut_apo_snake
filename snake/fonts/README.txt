
fonts folder        - simple bitmap font files

font_types.h            - simple bitmap fonts type definition
                        - simplified font type descriptor based on Microwindows/Nano-X library by Greg Haerr
        
font_prop14x16.c        - generated font files that defines a "Windows FreeSystem 14x16" Font
                        - includes bitmap representations of different characters, each with a defined height and width, represented by hexadecimal values

font_rom8x16.c          - generated font file that defines a "ROM 8x16 Font bios mode 12"
                        - includes bitmap representations of different characters, each with a defined height and width, represented by hexadecimal values
