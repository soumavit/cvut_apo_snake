
resources folder    - various game files, including a background image and leaderboard files related to a specific Snake game

snake.ppm               - ppm image file
                        - used as a loading screen when the game starts

leaderboard.txt         - text file
                        - stores current state of the in-game leaderboard

leaderboard_def.txt     - text file
                        - stores defaul state of the in-game leaderboard
